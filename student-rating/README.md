# Student Rating Frontend MVP

Фронтенд MVP корпоративной веб-системы геймифицированного рейтинга студентов.

## Что реализовано

- React + Vite + TypeScript.
- Публичный рейтинг студентов.
- Дашборд с карточками метрик и графиком.
- Локальное управление номинациями (добавление/удаление).
- Полностью автономный frontend без backend.

## Запуск

```bash
cd frontend
npm install
npm run dev
```

Открой `http://localhost:5173`.
